Available readers, sources, writers, filters and animation cues¶

Proxies generated for server side objects under simple Module

AMRConnectivity
one or more server manager objects.
AMRContour
one or more server manager objects.
AMRCutPlane This filter creates a cut-plane of the
AMRDualClip
one or more server manager objects.
AMRFragmentIntegration
one or more server manager objects.
AMRFragmentsFilter Combines the running of AMRContour, AMRFragmentIntegration, AMRDualContour and ExtractCTHParts
AMRGaussianPulseSource AMR dataset source, used for generating sample Berger-Collela AMR dataset with a Gaussian Pulse field at the center.
AMReXGridReader
This AMReX reader loads data stored in AMReX plt file format.
AMReXParticlesReader Reads particle data from AMReX plotfiles.
ANLHaloFinder
one or more server manager objects.
ANLSubhaloFinder
This filter assumes input similar to the output of the ANL Halo Finder’s first output port.
AVSUCDReader The AVS UCD reader reads binary or ASCII files stored in AVS UCD format.
AdaptiveCosmoCore The Adaptive Cosmo reader reads in multi-resolution, sampled binary data written in the LANL cosmo Format.
AddFieldArrays
Takes in an input data object and a filename.
AggregateDataset This filter aggregates a dataset onto a subset of processes.
AngularPeriodicFilter This filter generate a periodic multiblock dataset
AnimationCueBase Internal base interface for all animation cues.
AnimationScene Proxy for animation scene.
AnnotateAttributeData This filter can be used to add a text annotation to a Render View (or similar) using a tuple from any attribute array (point/cell/field/row etc.) from a specific rank (when running in parallel).
AnnotateGlobalData Annotate Global Data provides a simpler API for creating text annotations using vtkPythonAnnotationFilter.
AnnotateTime The Annotate Time source can be used to show the animation time in text annotation.
AnnotateTimeFilter The Annotate Time filter can be used to show the data time in a text annotation.
AppendArcLength

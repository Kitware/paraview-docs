1 # Adding support for making links between representations proxy or properties
2 
3 So far, the Link Editor allowed to create links for Views and Objects (ie.
4 pipeline sources) but Representations were not available. It is not the
5 case anymore: one can link representation proxies or representation properties.
6 For instance, one can link the Opacity property of two representations together
7 and then, when one is changed, the other is automatically changed the same way
8 and vice versa.

Incubator modules

Modules in this directory are incubating towards stability. Any Python wrappings appear under the paraview.incubator package. All API guarantees are disclaimed here. Modules may appear or disappear without affecting ParaView's stability guarantees.

Internal Usage

Only plugins may require incubator modules.

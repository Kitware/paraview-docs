1 ParaView 5.5.1 Release Notes
2 ============================
3 
4 ParaView 5.5.1 is a patch release. It addresses the following issues:
5 
6 
7 * The [Exodus library](https://github.com/gsjaardema/seacas/tree/master/packages/seacas/libraries/exodus)
8  in VTK has been upgraded to version 7.13. This version of the library contains a fix for a major issue encountered when loading side set geometry.
9 
10 * A bug in the **Ghost Cell Generator** that could lead to a segmentation fault
11 has been fixed.
12 
13 * Minor fixes to the camera model for the Cinema import and exporter have been added.
14 
15 * Filters that are not equipped to handle `vtkHyperTreeGrid` data sets return
16 an error rather than crash.
17 
18 * The `vtkXMLStructuredDataWriter` has been fixed to produce valid files when
19 appending time-varying data.
20 
21 * The SDK packaging for Linux has been fixed.
22 
23 The list of issues addressed by this release are available
24 [here](https://gitlab.kitware.com/paraview/paraview/milestones/11#tab-issues),
25 and merge requests included in this release are [here](https://gitlab.kitware.com/paraview/paraview/milestones/11#tab-merge-requests).

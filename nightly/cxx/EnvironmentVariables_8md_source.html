1 Environment Variables {#EnvironmentVariables}
2 =====================
3 
4 This page documents environment variables that affect ParaView behavior at
5 runtime.
6 
7 Variable | Description
8 ---------|---------------------------------------------------------
9 PARAVIEW_DATA_ROOT | Change the location of the data root for testing.
10 PV_DEBUG_PANELS | When set, debugging text will be printed out explaining the reason for creation of various widgets on the properties panel (pqPropertiesPanel).
11 PV_DEBUG_SKIP_OPENGL_VERSION_CHECK | Skip test to validate OpenGL support at launch.
12 PV_DEBUG_TEST | Prints debugging information about the testing framework during playback to cout.
13 PV_ICET_WINDOW_BORDERS | Force render windows to be 400x400 instead of fullscreen.
14 PV_DEBUG_REMOTE_RENDERING | Forces server-side render windows to swap buffers in order to see what is being rendered on the server ranks.
15 PV_MACRO_PATH | Additional directories defined by the user to store macros.
16 PV_ALLOW_BATCH_INTERACTION | Allow interactions in batch mode.
17 PV_PLUGIN_CONFIG_FILE | XML Plugin Configuration Files to specify which plugin to load on startup.
18 PV_PLUGIN_PATH | Directories containing plugins to be loaded on startup.
19 QT_MAC_NO_NATIVE_MENUBAR | Qt flag to force the Qt menu bar rather than the native mac menu bar.
20 PV_PLUGIN_DEBUG | (obsolete) Use **PARAVIEW_LOG_PLUGIN_VERBOSITY** instead.
21 PV_SETTINGS_DEBUG | (obsolete) Use **PARAVIEW_LOG_APPLICATION_VERBOSITY** instead.
22 PV_DEBUG_APPLY_BUTTON | (obsolete) Use **PARAVIEW_LOG_APPLICATION_VERBOSITY** instead.
23 
24 ParaView supports generating logs that includes debugging and tracking
25 information. The log messages are categorized and it is possible to temporarily
26 elevate the log level for any category using the following environment
27 variables. The value for each of the variables can be a number in the range
28 [-2, 9] or the strings `INFO`, `ERROR`, `WARNING`, `TRACE`, or `MAX` (See
29 `vtkLogger` for additional information about log levels).
30 
31 Variable | Description
32 ---------|-----------------------------------------
33 PARAVIEW_LOG_PLUGIN_VERBOSITY | Log messages related to ParaView plugins (see vtkPVLogger::GetPluginVerbosity())
34 PARAVIEW_LOG_PIPELINE_VERBOSITY | Log messages related to Pipeline execution (see vtkPVLogger::GetPipelineVerbosity())
35 PARAVIEW_LOG_DATA_MOVEMENT_VERBOSITY | Log messages related to data movement for rendering and other tasks (see vtkPVLogger::GetDataMovementVerbosity())
36 PARAVIEW_LOG_APPLICATION_VERBOSITY | Log messages related to the application (see vtkPVLogger::GetApplicationVerbosity())

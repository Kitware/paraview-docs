Detailed Description

Record log messages at or below a given verbosity on a particular ParaView process/rank.

This class can be used to record log messages at or below a verbosity specified by SetVerbosity(). If the process is run as an MPI job, the log messages from a rank enabled via SetRankEnabled() will be recorded.

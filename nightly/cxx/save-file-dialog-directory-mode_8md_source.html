1 ## Improved directory mode for save file-dialog
2 
3 When you save files into directories, ParaView presents a much cleaner and easy to understand dialog to select the output directory.
4 
5 When the file dialog is in directory mode (`pqFileDialog::Directory`):
6 1. The `File name` label has been renamed to `Directory name` and the unnecessary `Files of type` combo box
7 does not appear.
8 2. Although the files are visible, they are neither selectable nor editable (rename). This is useful for when you want to see the saved files. (ex: extracts)
9 
10 ### API Changes:
11 - The `pqFileDialogModel::isDir()` now has a `const` qualifier since it is used to set the appropriate flags for directories and files in the tree view. `QAbstractItemModel::flags()` is const qualified.

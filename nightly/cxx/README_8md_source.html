3 Introduction
4 ============
5 [ParaView][] is an open-source, multi-platform data analysis and
6 visualization application based on
7 [Visualization Toolkit (VTK)][VTK].
8 
9 The first public release was announced in October 2002. Since then, the project
10 has grown through collaborative efforts between [Kitware Inc.][Kitware],
11 [Sandia National Laboratories][Sandia],
12 [Los Alamos National Laboratory][LANL],
13 [Army Research Laboratory][ARL], and various other
14 government and commercial institutions, and academic partners.

47 Building
48 ========
49 
50 There are two ways to build ParaView:
51 
52 * Perhaps the easiest method for beginners to build ParaView from source is
53 using the [ParaView Superbuild][sbrepo]. The superbuild downloads and builds all
54 of ParaView's dependencies as well as ParaView itself.
55 
56 * It is also possible to [build ParaView][build] without using the superbuild.
57 ParaView's dependencies must be available on the system.
58 
59 [sbrepo]: https://gitlab.kitware.com/paraview/paraview-superbuild
60 [build]: Documentation/dev/build.md

62 Reporting Bugs
63 ==============
64 
65 If you have found a bug:
66 
67 1. If you have a source-code fix, please read the [CONTRIBUTING.md][] document.
68 
69 2. Otherwise, please join the [ParaView Discourse][] forum and ask about
70  the expected and observed behaviors to determine if it is really a bug.
71 
72 3. Finally, if the issue is not resolved by the above steps, open
73  an entry in the [ParaView Issue Tracker][].
74

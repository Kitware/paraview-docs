1 ## Add Conduit Node IO and Catalyst Replay Executable ##
2 
3 To assist in debugging in-situ pipelines, Catalyst now
4 supports `conduit_node` I/O. The `params` argument to each invocation of
5 `catalyst_initialize`, `catalyst_execute`, and `catalyst_finalize` can be
6 written to disk. These can later be read back in using the `catalyst_replay`
7 executable. See the corresponding documentation:
8 https://catalyst-in-situ.readthedocs.io/en/latest/catalyst_replay.html

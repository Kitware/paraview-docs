Detailed Description

This header contains the definition of the dispatch table.

This table contains a pointer to every netcdf function. When a file is opened or created, the dispatch code in libdispatch decides which dispatch table to use, and all subsequent netCDF calls for that file will use the selected dispatch table. There are dispatch tables for HDF5, HDF4, pnetcdf, etc.

Author
Dennis Heimbigner, Ed Hartnett

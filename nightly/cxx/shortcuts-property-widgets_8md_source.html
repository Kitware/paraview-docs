1 # Keyboard shortcuts and property widgets
2 
3 ParaView now has a singleton object, `pqKeySequences`, that
4 can be used to create and manage Qt shortcuts so that there
5 are no collisions between widgets/actions that both want the
6 same keyboard shortcut.
7 
8 An example of this is when a property panel might wish to
9 use both a handle widget and a line widget. Each of these
10 registers shortcuts for the P and Ctrl+P/Cmd+P keys.
11 Because multiple widgets registered shortcuts for the same
12 key, Qt never signals *either* widget unambiguously when
13 these keys are pressed.
14 
15 By registering shortcuts with this class, widgets will have
16 their shortcuts enabled and disabled so that only one is
17 using the shortcut at a time. The user is shown which widgets
18 have shortcuts registered (via a frame highlighted with a
19 thin blue line, the same as the active view's highlighting)
20 and can enable/disable widgets by clicking on the Qt widget's
21 background.
22 
23 ## Developer-facing changes
24 
25 To accomplish the above, several changes to pqPropertyWidget
26 and its subclasses as well as to the pqPointPickingHelper
27 were required:
28 
29 + `pqPropertyWidget` inherits QFrame instead of QWidget.
30  By default, the frame is configured not to render or
31  take any additional space. However, when shortcuts are
32  registered with `pqKeySequences`, a new decorator class
33  named `pqShortcutDecorator` configures the frame to
34  reflect when shortcuts are registered and enabled.
35 + `pqPointPickingHelper` previously accepted any QObject
36  as a parent. However, all occurrences of this class used
37  property widgets as the parent. By forcing parents to be
38  of this type, subclasses of `pqInteractivePropertyWidget`
39  that used the picking helper do not need to manage visibility
40  changes. (Instead `pqShortcutDecorator` enables/disables
41  shortcuts to match visibility changes.)
42  If you use `pqPointPickingHelper` in your code, ensure its
43  parent class is a `pqPropertyWidget` and remove any usage
44  of the `setShortcutEnabled()` method.

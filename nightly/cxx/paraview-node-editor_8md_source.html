5 ### Overview
6 
7 This plugin contains a node editor for ParaView that makes it possible to conveniently modify filter/view properties, filter input/output connections, as well as the visibility of outputs in views.
8 The editor is completely compatible with the existing ParaView widgets such as the pipeline browser and the properties panel (one can use them simultaneously).

10 ### User Manual
11 
12 * Filters label :
13  - Left click : select filter as active source
14  - Left click + CTRL : add filter to the current selection
15  - Middle click : Delete the filter
16  - Right click : toggle between invisible, normal and invisible mode for the filter properties
17 * Filters input ports :
18  - Left click + CTRL : set all selected output ports as input. If only one input is accepted it will use the last selected object as input.
19  - Middle click : Clear all incoming connections for this port
20 * Filters output ports :
21  - Left click: set output port as active selection
22  - Left click + CTRL: add output port to current selection
23  - Left click + SHIFT: toggle visibility in active view
24  - Left click + SHIFT + CTRL: hide all but this port in active view
25 * View label :
26  - Left click : select as active view
27  - Right click : increment verbosity
28 * Drag and drop from a port to an other to create links between 2 filters. This is not supported between a view node and a filter yet.

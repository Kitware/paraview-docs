1 # Add Threshold Table Filter to ParaView
2 
3 Add a table thresholding filter to the ParaView interface. The *Threshold Table* filter can now be found under *Filters* > *Data Analysis* in order to perform four different thresholding operations. Its input parameters are :
4 
5 - **Column** : Specifies which column to use for thresholding.
6 - **Min Value** : Lower bound of the thresholding operation.
7 - **Max Value** : Upper bound of the thresholding operation.
8 - **Mode** : Specifies one of the four available thresholding methods.
9 
10 The four different modes are :
11 
12 - **Below Max Threshold** : Accepts rows with values < MaxValue
13 - **Above Min Threshold** : Accepts rows with values > MinValue
14 - **Between** : Accepts rows with values > MinValue and < MaxValue
15 - **Outside** : Accepts rows with values < MinValue and > MaxValue

1 # zSpace plugin
2 
3 A new Microsoft Windows only plugin named 'zSpace' has been added. It adds
4 a new view named 'zSpaceView' that let a user interact with a zSpace
5 device directly. This device is designed to work with Crystal Eyes
6 stereo, in full screen or in a cave display. The zSpace glasses
7 use a head tracking system that allow the user to look around 3D
8 object by moving his head.
9 
10 This plugin requires a [zSpace System Software >= 4.4.2](https://support.zspace.com/s/article/zSpace-System-Software-release-Required?language=en_US),
11 and a SDK version >= 4.0.0.
12 It was tested on a zSpace 200 device but should be compatible with more recent devices as well.

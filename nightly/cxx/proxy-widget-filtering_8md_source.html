1 ## Improvements to pqProxyWidget
2 
3 `pqProxyWidget` is the widget used to auto-generate several panels based on
4 proxy definitions. The class supports two types of panel visibilities for
5 individual widgets for properties on the proxy: **default** and **advanced**.
6 These needed to match the value set for the `panel_visibility` attribute used
7 the defining the property in ServerManager XML. `pqProxyWidget` now has API to
8 make this configurable. One can now use arbitrary text for `panel_visibility`
9 attribute and then select how that text is interpreted, default or advanced, by
10 a particular `pqProxyWidget` instance. This makes it possible for proxies to
11 define properties that are never shown in the **Properties** panel, for example,
12 but are automatically shown in some other panel such as the **Multiblock
13 Inspector** without requiring any custom code. For more details, see
14 `pqProxyWidget::defaultVisibilityLabels` and
15 `pqProxyWidget::advancedVisibilityLabels`.

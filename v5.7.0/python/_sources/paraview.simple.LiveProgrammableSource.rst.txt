paraview.simple.LiveProgrammableSource
======================================

.. currentmodule:: paraview.simple

.. autofunction:: LiveProgrammableSource

Data Descriptors
----------------


   
   OutputDataSetType
       The value of this property determines the dataset type
       for the output of the programmable source.
   
   PauseLiveSource
       Pause live source.
   
   PythonPath
       A semi-colon (;) separated list of directories to add to
       the python library search path.
   
   Script
       This property contains the text of a python program that
       the programmable source runs. To execute external python scripts on the ParaView server, use:
       execfile('full_path/script_name.py'). Documentation is found in the Programmable Filter
       chapter of the ParaView Guide.
   
   ScriptCheckNeedsUpdate
       Specify the script to use to check if the filter/source has new data.
       This is only needed for "live" sources, that can cause data to be available
       outside of ParaView.
   
   ScriptRequestInformation
       This property is a python script that is executed during
       the RequestInformation pipeline pass. Use this to provide information
       such as WHOLE_EXTENT to the pipeline downstream.
   
   TimestepValues
       Available timestep values.

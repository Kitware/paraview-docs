1 # Bounding Ruler can now measure axes along oriented bounding boxes
2 
3 In addition to measuring the X, Y, or Z axis of the axis-aligned bounding box
4 of a data set, the **Bounding Ruler** filter can now measure the Major, Medium,
5 and Minor axes of the data set's oriented bounding box. These axes are new options
6 in the **Axis** property of the **Bounding Ruler** filter. As an example, one
7 can use ParaView's selection capabilities to extract two points of interest and
8 then apply the **Bounding Ruler** with the **Axis** property set to
9 **Oriented Bounding Box Major Axis** to measure how the distance between these
10 points changes in time.
11 
12 One caveat to these options is that all the points in a data set need to be copied
13 to a single rank to compute the oriented bounding box. When used in a parallel
14 server setting, this may be slow or lead to memory exhaustion if the data set does
15 not fit onto one rank.

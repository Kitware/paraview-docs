function vtk_hash_source ( )

Generate a header containing the hash of a file.

Add a rule to turn a file into a MD5 hash and place that in a C string.

vtk_hash_source(
INPUT <input>
[NAME <name>]
[ALGORITHM <algorithm>]
[HEADER_OUTPUT <header>])

The only required variable is INPUT.

  • INPUT: (Required) The path to the file to process. If a relative path is given, it will be interpreted as being relative to CMAKE_CURRENT_SOURCE_DIR.
  • NAME: This is the base name of the header file that will be generated as well as the variable name for the C string. It defaults to basename of the input suffixed with Hash.
  • ALGORITHM: This is the hashing algorithm to use. Supported values are MD5, SHA1, SHA224, SHA256, SHA384, and SHA512. If not specified, MD5 is assumed.
  • HEADER_OUTPUT: the variable to store the generated header path.

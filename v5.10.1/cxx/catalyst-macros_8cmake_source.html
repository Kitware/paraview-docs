19 
20 The `TARGET` identifies the target to create in this function. This function
21 creates source files to adapt to the Catalyst ABI and creates a header for use
22 in the actual implementation.
23 
24 `LIBRARY_DESTINATION` defaults to `${CMAKE_INSTALL_BINDIR}/catalyst` on Windows
25 and `${CMAKE_INSTALL_LIBDIR}/catalyst` elsewhere and indicates where the
26 library will be placed in the build and install tree.
27 
28 If the `catalyst::catalyst` target is not available in the usage location, an
29 alternative available target name may be given by the `CATALYST_TARGET`
30 argument.
31 
32 The library will be installed with the export set provided to `EXPORT` if
33 given.

Detailed Description

property-widget for creating selection queries.

pqSelectionQueryPropertyWidget is a pqPropertyWidget subclass that can be used for a property accepting a string representing a selection query. The widget allows users to build selection queries easily.

To use this widget on a property with vtkSMSelectionQueryDomain, set the panel_widget to selection_query (

See also
pqStandardPropertyWidgetInterface).

Caveats

Currently, this widget can faithfully present GUI for query strings that follow a specific form that is generated by this property itself. Any other forms, though valid queries, will not be represented properly in the GUI.

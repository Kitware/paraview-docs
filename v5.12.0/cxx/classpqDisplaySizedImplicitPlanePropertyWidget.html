Detailed Description

pqDisplaySizedImplicitPlanePropertyWidget is a custom property widget that uses "DisplaySizedImplicitPlaneWidgetRepresentation" to help users interactively set the origin and normal for a plane.

To use this widget for a property group (vtkSMPropertyGroup), use "InteractivePlane" as the "panel_widget" in the XML configuration for the proxy. The property group should have properties for following functions:

  • Origin: a 3-tuple vtkSMDoubleVectorProperty that will be linked to the origin of the interactive plane.
  • Normal: a 3-tuple vtkSMDoubleVectorProperty that will be linked to the normal for the interactive plane.
  • Input: (optional) a vtkSMInputProperty that is used to get data information for bounds when placing/resetting the widget.

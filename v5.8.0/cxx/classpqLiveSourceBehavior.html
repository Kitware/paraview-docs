Detailed Description

pqLiveSourceBehavior adds support for "live" algorithms.

These are vtkAlgorithm subclasses that have a method GetNeedsUpdate which returns true (bool) when source may have new data and should be refreshed.

To indicate a source is a "live source", one needs to simply add <LiveSource> XML hint for the source proxy.

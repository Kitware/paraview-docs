Collects OpenGL information from the object.

object must be either a vtkPVView or a vtkRenderWindow. If not, this call will create a vtkRenderWindow temporarily and use it to obtain OpenGL capabilities information (see vtkPVRenderingCapabilitiesInformation::NewOffscreenRenderWindow()).

Detailed Description

adds support for optionally reading image stacks.

vtkImageFileSeriesReader is designed for vtkImageReader2 and subclasses. This adds API to optionally treat the file series as an image stack rather than a temporal dataset. When ReadAsImageStack is true, we simply by-pass the superclass and instead pass all filenames to the internal reader and then let it handle the pipeline requests.

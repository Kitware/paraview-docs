Detailed Description

access proxies that define catalyst export state

vtkSMExportProxyDepot is a container for export proxies. These include Writer proxies, SaveScreenShot proxies and the global Catalyst Options Proxy. It is a helper class for the SMSessionProxyManager intended to standardize an API for working with them.

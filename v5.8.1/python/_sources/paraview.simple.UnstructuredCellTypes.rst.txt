paraview.simple.UnstructuredCellTypes

.. currentmodule:: paraview.simple

Data Descriptors
----------------


   
   Addnodestocompletepolynomialbasis
       When true, some quadratic Lagrange cell types will have additional nodes.
       
       Specifically,
       triangles will have 7 points instead of 6;
       tetrahedra will have 15 points instead of 10; and
       wedges will have 21 points instead of 18.
       These additional points increase the number of polynomial terms
       in each element so that the shape functions can span a larger
       space of polynomial functions.
   
   BlocksDimensions
       Sets the number of blocks of cells in each direction.
       A cell block may be divided into multiple cells (e.g. there will be
       6 pyramid cells in a cell block). If a 1D cell type is chosen only
       the first blocks dimension will be used. If a 2D cell type is chosen the
       first and second blocks dimensions will be used.
   
   CellOrder
       This property specifies the interpolation order for Lagrange cell types.
   
   CellType
       This property specifies the type of cells to generate.
   
   OutputPrecision
       This property specifies the precision of point locations and fields.
   
   PolynomialFieldOrder
       This property specifies the order for the "Polynomial" point data field.

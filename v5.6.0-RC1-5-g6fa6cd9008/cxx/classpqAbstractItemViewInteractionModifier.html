Detailed Description

modifier to extend interaction with Qt views.

pqAbstractItemViewInteractionModifier adds support for ParaView specific interactions to standard pqAbstractItemViewInteractionModifier subclasses such as QTreeView or QTableView. It's designed to work wtih QAbstractItemView and hence should work for QTableWidget, QTreeWidget, etc. as well.

It adds support for following modes of user interaction:

  • 1. For checkable items, Qt expects the users to click on the checkbox. ParaView users have said that that's confusing and they would rather check-state toggled when clicking on the entire item, not just the checkbox icon. This class adds support for that.
  • 2. Add support to change check state for all items in the current selection. This makes it easy to modify check states for multiple items.
  • 3. Add a context menu to enable toggling of check state for selected items as well as sorting, if enabled for the view.

1 # Add a ShaderReplacements property to the Geometry representation
2 
3 This new property allows the user to customize the OpenGL shaders by providing
4 some replacement strings for the shader produced by VTK. The replacements are
5 provided through a Json string like this:
6 
7 ```json
8 [
9  {
10  "type": "fragment",
11  "original": "//VTK::Light::Impl",
12  "replacement": "gl_FragData[0]=vec4(1.0,0.0,0.0,1.0);\n"
13  },
14  // other replacements can follow
15 ]
16 ```
17 
18 In the UI, the property is represented with a custom widget that allows to load
19 Json from a file and save its path as a preset which is saved in the SMSettings
20 (thus in user's configuration file). It is possible to select a preset, remove
21 it from the list and edit the content of the Json string in a text edit widget.
